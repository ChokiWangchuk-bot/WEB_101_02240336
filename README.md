## What is this project?
This is a TikTok clone built as a practical web development project.
I used Next.js and Tailwind CSS to replicate core TikTok features.

## What I Built
- A video feed on the home page
- Navigation links on the sidebar
- Following, Explore, Live, Upload, and Profile pages
- Login and Signup pages with form validation and error message handling

## Tools and Technologies Used
- Next.js – For building the web application
- Tailwind CSS – For styling and layout
- React Icons – For sidebar icons
- React Hook Form – For managing login and signup form state

## How to Run the Project

### Step 1: Clone the repository
git clone https://github.com/ChokiWangchuk-bot/WEB_101_02240336.git

### Step 2: Navigate to the project folder
cd WEB_101_02240336

### Step 3: Install dependencies
npm install

### Step 4: Start the development server
npm run dev

### Step 5: Open in your browser
http://localhost:3000
